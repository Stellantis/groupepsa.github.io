### `Radio.Text()`

| **Description** | Return the radio text.
| **Response** | *String* Radio text.
| **Parameter**   | *Void*

#### Example

```javascript
// Get the Radio text
var RadioText = Radio.Text()
// RadioText = "Thriller by Michael Jackson"
```
