### `Radio.PSName()`

| **Description** | Return the program service name..
| **Response** | *String*  Program service name.
| **Parameter**   | *Void*

#### Example

```javascript
// Get the Radio Program Service Name
var RadioPSName = Radio.PSName()
// RadioPSName = "BBC Radio"
```
